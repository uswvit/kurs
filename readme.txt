Примечания:
1. Сверстано под ПК, на футер не хватило времени, для малых разрешений вообще ничего не успел.

2. Поведение верхней полоски меню с position:sticky не до конца правильное - скролится только до блока, в котором лежит, а не на всем сайте. 

3. Позиционирование картинки в хэдере справа не такое, как в Figma. При копировании стилей оттуда получается отступ справа, подбирал сам в итоге. Заодно разобрался с z-index по всему макету, где эта картинка перекрывала элементы из-за своей позиции.

4. В блоке с тремя ценами сделал ролловер, убив немало времени - особенности синтаксиса SCSS пришлось активно гуглить, т.к. для псевдоклассов объявление идет через амперсанд. Но разобрался и сделал. Из недоступного для понимания - фон и цвет шрифта меняются, карточка и кнопка сделаны миксинами, экспериментировал с вложенностью миксинов, но кнопка все равно только при наведении на нее меняет цвет, а не при наведении на сам блок. Оставил только рабочий вариант.

5. Последний блок сделал на grid. Возникла сложность с наложением тени, отказался от нее из-за времени.

6. Вложенность в SCSS использовал может и не полностью правильно, вижу избыточность, но как есть. Очень понравился инстумент, заморочки с npm не прошли напрасно.
